# Codex Multi Login

[English](README.md) | [한국어](README_KO.md)

여러 Codex 계정을 쓰는 사람을 위한 VS Code 확장입니다.
계정을 한곳에 모아 두고, 원하는 계정으로 바꾸기 쉽게 도와줍니다.

## 할 수 있는 일

- Codex 계정 추가
- 활성 계정 바꾸기
- quota 다시 확인하기
- 계정 정보 저장하기
- 간단한 대시보드 열기

## 사용 방법

1. 확장을 설치합니다.
2. 대시보드를 엽니다.
3. 필요하면 계정을 추가합니다.
4. 원하는 계정으로 바꿉니다.
5. quota를 확인합니다.

## 참고

- 일부 기능은 데스크톱 VS Code에서만 됩니다.
- 계정 정보는 VS Code 저장소에 보관됩니다.
- 자세한 설명은 개발자 문서를 보세요.
